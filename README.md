# nodejs-web-vite-pnpm-lit-template

## Description
A lit template for the vite packager.

## Tech stack
- pnpm
- npm
- vite
- lit

## Docker stack
- alpine

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
